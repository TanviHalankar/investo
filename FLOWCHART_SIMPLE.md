## Key Components

### Services
- **UserDataService**: Manages user data (portfolio, watchlist, lessons)
- **PortfolioService**: Handles portfolio operations
- **RealTimeService**: WebSocket connection for live stock data
- **GuideService**: In-app tips and guidance system

### Screens
1. **OnboardingScreen**: First-time user introduction
2. **LoginScreen**: User authentication
3. **RegisterScreen**: New user registration
4. **HomeScreen**: Main dashboard
5. **PortfolioScreen**: Portfolio management
6. **LearningScreen**: Educational content
7. **ProfileScreen**: User profile and settings
8. **LeaderboardScreen**: Global rankings
9. **ChatScreen**: AI chatbot with owl character
10. **IPOScreen**: IPO listings
11. **PredictionScreen**: Stock predictions

### Data Storage
- **Local**: SharedPreferences (offline access)
- **Remote**: Firestore (cloud sync)
- **Real-time**: WebSocket (live updates)
